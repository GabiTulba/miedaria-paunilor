# Miedﾄビia Pﾄブnilor - Official Website

This project is the official website for Miedﾄビia Pﾄブnilor, a traditional meadery. The website will serve as a digital storefront and information hub, showcasing our unique mead products, sharing our story, and providing a way for customers to get in touch.

Built with modern web technologies, this site aims to be fast, responsive, and user-friendly.

## Tech Stack

This project is built using the following technologies:

*   **Framework**: [Next.js](https://nextjs.org/) 14+
*   **Library**: [React](https://reactjs.org/) 18+
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/) (or other modern CSS framework)
*   **Package Manager**: `npm` 10+ or `yarn` 1.22+
*   **Node.js**: v20.x or later
